# Python introduction

## Python variables

What are variables?
- The shorthand for variables is var
- They store data of different types

Types of variables:
- int
- float
- boolean
- string

## Operators

There are two types of operators. 

### Arithmetic operators:
- ```+``` 
- ```-```
- ```*```
- ```/```

### Comparison operators:
- `<`
- `>`
- `==`
- `!=`
- `<=`
- `>=`
